# Phase 2 Report (Snapshot)

This is a curated snapshot of the Phase 2 implementation and deployment status for archival under assets.

- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Auth: JWT + bcrypt
- File support: Cloudinary
- Email: SMTP (Nodemailer)
- Deployment: Vercel + Render
